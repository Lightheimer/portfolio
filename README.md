Portfolio personnel. Software engineer · Laravel · Next.js · ICT4D.

## Structure

- `Analyse/` — brief, recherches, sources, dogmes appliqués.

## Direction artistique

Cartier-Serenity (canon chaude). Ivoire `#F8F6F1`, noir `#0A0A0A`, doré `#B8935A`, cendre `#E8E4DC`. Polices Fraunces (display) + Inter (texte). Light-first, dark optionnel discret.

Référence dogme : `/memories/serenity-design-principles.md`.

## Démarrer

```bash
cd application
pnpm install
pnpm dev
```

Puis http://localhost:3000.
