# Gym Management — Spring Boot

Web application for managing a gym (members, subscriptions, sessions).
University project built during the Software Engineering programme at **IAI-TOGO**.

## Stack

- **Language** — Java 17
- **Framework** — Spring Boot, Spring MVC, Spring Data JPA
- **Database** — MySQL
- **Views** — Thymeleaf
- **Build** — Maven

## Scope

- Member registration and profile management
- Subscription plans (monthly, quarterly, yearly)
- Session check-in
- Admin dashboard

## Status

Academic project — kept as a reference of MVC architecture and Spring Boot fundamentals.
