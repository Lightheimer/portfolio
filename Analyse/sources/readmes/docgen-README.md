# DocGen

Document generator. Produces clean PDF documents — monthly activity reports, invoices, attestations, timesheets — from structured input.

## Stack

- **Runtime** — Node.js
- **Language** — JavaScript
- **PDF engine** — HTML + headless Chromium pipeline
- **Templates** — HTML / CSS

## Why

Repetitive document production (monthly reports, invoices) deserved a small reliable tool — typed input, typed output, no manual formatting.

## Status

Active. Used to produce internal reports during the **UNICEF Togo ICT4D internship** (monthly activity reports).
